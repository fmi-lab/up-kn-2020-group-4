# Recursion

- A <b>recursive function</b> in C++ is a function that calls itself. 

- Recursive termination conditions: 
```
	void countDown(int count) {
    	std::cout << "push " << count << '\n';
    	if (count > 1) { // termination condition
        	countDown(count-1);
    	} 
		std::cout << "pop " << count << '\n';
	}
```
